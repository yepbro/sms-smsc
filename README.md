## SMSC

Пакет для работы с API smsc.ru по https протоколу.

### Документация

https://yepbro.ru/packages/sms/smsc

### Установка

Стандартная установка (с guzzlehttp/guzzle:^7.8)

```bash
composer require yepbro/sms-smsc
```
